A **Casa Hacker** é uma [organização sem fins lucrativos](https://www.casahacker.org/) criada com a missão de fazer das tecnologias e a inovação social um lugar para todos. Nossa história mostra que periferia, tecnologia, criatividade e oportunidades justas desafiam estruturas de desigualdades. Hoje, desenvolvemos programas de desenvolvimento de lideranças e educação STEAM da base para a base, de periferia para a periferia.

Para saber mais sobre nosso impacto ou fazer parte, visite nosso site: [casahacker.org](https://casahacker.org).
